## 🎓 Entwickler-Notizen

### CSS-Variablen
Alle Farben und Werte sind als CSS-Variablen in `:root` definiert und können einfach angepasst werden:

```css
--primary-color: #d4af37;  /* Gold */
--secondary-color: #000000; /* Schwarz */
--text-dark: #ffffff;       /* Weißer Text */
--text-light: #b0b0b0;      /* Grauer Text */
```

### JavaScript-Module
Der JavaScript-Code ist modular aufgebaut und kann leicht erweitert werden.

### Formular-Integration
Die Formulare sind vorbereitet für Backend-Integration. Aktuell verwenden sie `setTimeout()` für Demo-Zwecke.
