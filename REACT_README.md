# Dushi's Coiffeur - React Website

Willkommen bei der React-Version der Website von Dushi's Coiffeur, einem Friseursalon in Winterthur.

## 🚀 Installation

1. Installieren Sie die Abhängigkeiten:
```bash
npm install
```

## 💻 Entwicklung

Starten Sie den Entwicklungsserver:
```bash
npm start
```

Die Website öffnet sich automatisch unter [http://localhost:3000](http://localhost:3000).

## 🏗️ Build für Produktion

Erstellen Sie eine produktionsreife Version:
```bash
npm run build
```

Die optimierten Dateien werden im `build`-Ordner erstellt.

## 📁 Projektstruktur

```
src/
├── components/          # Wiederverwendbare Komponenten
│   ├── Navbar.js       # Navigation
│   └── Footer.js       # Footer
├── pages/              # Seitenkomponenten
│   ├── Home.js         # Startseite
│   ├── Services.js     # Dienstleistungen
│   ├── About.js        # Über Uns
│   ├── Contact.js      # Kontakt
│   ├── Booking.js      # Termin buchen
│   ├── Impressum.js    # Impressum
│   └── Datenschutz.js  # Datenschutz
├── App.js              # Haupt-App-Komponente mit Routing
├── index.js            # Entry Point
└── index.css           # Globale Styles
```

- ✅ Alle originalen Inhalte und Designs beibehalten

## 📱 Seiten

- **Home** - Startseite mit Hero-Section und Service-Übersicht
- **Dienstleistungen** - Vollständige Preisliste
