### Technische Anforderungen:

- **Format:** JPG oder PNG
- **Größe:** Mindestens 800x800 Pixel
- **Seitenverhältnis:** 1:1 (quadratisch)
- **Qualität:** Hochauflösend, professionell
- **Dateigröße:** Optimiert für Web (max. 500KB pro Bild)
- **Stil:** Einheitlicher Hintergrund, professionell, freundlich

### Empfohlener Aufnahmestil:

- Heller, neutraler Hintergrund
- Gute Beleuchtung
- Professionelle Kleidung
- Freundlicher, offener Gesichtsausdruck
- Hochformat/Portrait-Orientierung
- Fokus auf Gesicht und Oberkörper
