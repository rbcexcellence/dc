# Gallery Images Placeholder

## 🖼️ Galerie-Bilder benötigt

Für die Galerie-Sektion auf der Startseite werden hochwertige Fotos von Frisur-Arbeiten benötigt.

### Bildkategorien:

1. **Damenschnitte** - Moderne und klassische Haarschnitte für Damen
2. **Balayage/Färbungen** - Farbarbeiten, Strähnchen, Ombre
3. **Herrenschnitte** - Klassische und moderne Männerfrisuren
4. **Hochsteckfrisuren** - Elegante Frisuren für Events
5. **Fashion Colors** - Kreative, trendige Farben
6. **Kurzhaar-Styles** - Pixie Cuts, Bobs, etc.

### Technische Anforderungen:

- **Format:** JPG (bevorzugt) oder PNG
- **Größe:** Mindestens 1200x1200 Pixel
- **Seitenverhältnis:** 1:1 (quadratisch) für einheitliches Grid
- **Qualität:** Hochauflösend, scharf, gute Beleuchtung
- **Dateigröße:** Optimiert für Web (200-800KB pro Bild)
- **Stil:** Professionell, konsistent

### Empfohlener Fotostil:

#### Lighting & Setup
- Helle, gleichmäßige Beleuchtung
- Neutraler/weicher Hintergrund
- Fokus auf der Frisur
- Professionelle Kameraqualität

#### Komposition
- Zentrierter Fokus auf die Frisur
- Verschiedene Winkel (vorne, seite, hinten)
- Nahaufnahmen für Details
- Model mit freundlichem Ausdruck

#### Standards
- Konsistente Farbgebung
- Gute Schärfe und Kontrast
- Keine starken Filter
- Natürliche Hautfarben

### Verwendung auf der Website:

Die Galerie-Sektion ist vorbereitet für:
- Grid-Layout (responsive)
- Lightbox-Funktionalität (zukünftig)
- Filter nach Kategorien
- Lazy Loading für Performance

### HTML-Integration:

Aktuell verwendet die Website Platzhalter. Für echte Bilder:

```html
<div class="gallery-item" data-category="damen">
  <img src="images/gallery/damen-schnitt-1.jpg" 
       alt="Moderner Damenschnitt" 
       loading="lazy">
</div>
```

### Best Practices:

✅ **Qualität vor Quantität** - Lieber weniger, aber hochwertige Bilder
✅ **Kundeneinwilligung** - Zuständigung zur Veröffentlichung einholen
✅ **Vielfalt zeigen** - Verschiedene Stile, Altersgruppen, Haartypen
✅ **Saisonal aktualisieren** - Regelmäßig neue Arbeiten hinzufügen
✅ **Wasserzeichen** - Optional: Diskretes Logo für Markenschutz

### Rechtliche Hinweise:

⚠️ **Wichtig:**
- Schriftliche Einwilligung der Models zur Veröffentlichung
- DSGVO-konform speichern
- Keine Kinderfotos ohne elterliche Zustimmung
- Urheberrechte beachten
